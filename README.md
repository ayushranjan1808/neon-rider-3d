# Neon Rider 3D 🏎️💨

A high-octane, browser-based 3D cyberpunk racing game built entirely with React, Three.js, and React Three Fiber. Experience unbound arcade physics, procedurally generated cityscapes, and seamless cross-platform controls right in your browser.

## ✨ Features

* **Procedural World Generation:** Endless, dynamically generated track and blocky city environments so no two races look exactly the same.
* **Custom Kinematic Physics:** Built from scratch without heavy physics engines for snappy, arcade-style drifting and handling (inspired by *Asphalt*).
* **Mobile & Cross-Platform:** Fully responsive UI with on-screen touch controls.
* **Tilt-to-Steer:** Utilizes the `DeviceOrientation` API for immersive gyro steering on mobile devices.
* **Cinematic UI:** High-quality "reveal zoom" menu animations, minimap navigation, and Tailwind-styled HUD.
* **Progression System:** Nitro boosts, collectible items, and game state persistence using `Zustand` and `LocalStorage`.

## 🛠️ Tech Stack

* **Core:** React, TypeScript, Vite
* **3D Rendering:** Three.js, React Three Fiber (R3F)
* **State Management:** Zustand
* **Styling:** Tailwind CSS

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
```bash
   git clone [https://github.com/ayushranjan1808/neon-rider-3d.git](https://github.com/ayushranjan1808/neon-rider-3d.git)
Navigate to the project directory:

Bash
   cd neon-rider-3d
Install dependencies:

Bash
   npm install
Start the development server:

Bash
   npm run dev
Open your browser and visit http://localhost:5173

🎮 Controls
Desktop
W / Up Arrow: Accelerate

S / Down Arrow: Brake / Reverse

A / Left Arrow: Steer Left

D / Right Arrow: Steer Right

Shift: Nitro Boost

Mobile
Touch Interface: Use on-screen UI buttons for acceleration, braking, and nitro.

Gyro Steer: Tilt your device left and right to steer the car.
